# Dota 2 Mid-Lane Matchup Tracker

A beginner Python project for tracking Dota 2 mid-lane matchups, win rates, difficulty, and personal gameplay notes.

## Project Overview

I play mid lane in Dota 2, mainly heroes like Puck and Queen of Pain. This project helps me record matchups, track wins and losses, calculate win rates, and store notes about how each matchup should be played.

The goal of this project is to practice basic programming concepts such as:

- Classes and objects
- Methods
- Lists and dictionaries
- Simple calculations
- Clean project organization
- Basic data analysis

## Example Matchups

Some matchups I want to track:

- Puck vs Huskar
- Puck vs Storm Spirit
- Queen of Pain vs Invoker
- Queen of Pain vs Keeper of the Light
- Queen of Pain vs Storm Spirit

## Features

Current planned features:

- Create a matchup between my hero and an enemy mid hero
- Add wins and losses
- Calculate total games
- Calculate win rate
- Store matchup difficulty
- Store personal notes
- Print a summary of the matchup
